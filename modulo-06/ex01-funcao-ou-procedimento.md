    Str() — Função: Retorna um valor do tipo texto resultante da conversão do número informado.

    QOut() — Procedimento: Executa a ação de exibir uma informação no console, sem devolver um dado para ser utilizado pelo restante do código.

    Date() — Função: Consulta o sistema operacional e retorna o valor correspondente à data atual para o programa.

    Len() — Função: Avalia a estrutura informada e retorna um valor numérico que representa a quantidade de seus elementos ou caracteres.

    Upper() — Função: Processa a string original e retorna uma nova string com todas as letras convertidas para maiúsculas.

    ClearScreen() — Procedimento: Apenas executa a ação visual e estrutural de limpar o monitor, não devolvendo nenhum valor ao fluxo de execução.
